# Amazon E-Commerce Analysis (Power BI Project)

## 📌 Project Overview
This project presents a comprehensive analysis of an Amazon e-commerce dataset to derive insights into revenue performance, customer behavior, and product demand. The analysis leverages SQL, Excel, and Power BI to transform raw data into meaningful insights and interactive visualizations that support data-driven decision-making.

## 📊 Key Highlights
- **Data Preparation**: Cleaned and structured 1,000+ records to ensure data accuracy and consistency.  
- **SQL Analysis**: Developed queries to extract key performance indicators, including sales trends, return rates, and category performance.  
- **Dashboard Development**: Designed an interactive Power BI dashboard featuring:
  - Revenue trends and seasonal patterns  
  - Regional and category-wise performance  
  - Customer segmentation based on loyalty tiers  
  - Sales and return behavior analysis  
- **Insight Generation**: Identified high-value customer segments (Platinum tier), peak sales periods (May–June), and critical areas for improvement, including a 27% return rate.

## 🛠 Tools & Technologies
- SQL (MySQL Workbench)  
- Microsoft Excel (Data validation and preprocessing)  
- Power BI (Data visualization, DAX, dashboard design)  

## 📁 Repository Structure
- `Dashboard/` → Power BI `.pbix` file  
- `Data/` → Dataset used for analysis  
- `SQL/` → SQL scripts for KPI extraction  
- `Documentation/` → Detailed report and presentation  

## 🚀 Business Outcomes
- Identified Platinum customers as the most valuable segment despite lower volume.  
- Highlighted May–June as peak revenue periods, enabling targeted marketing strategies.  
- Provided actionable recommendations to optimize customer retention, reduce return rates, and improve regional performance.
