## 🚀 Project Overview  
This project delivers a comprehensive analysis of an Amazon e-commerce dataset to uncover insights into **revenue performance, customer behavior, and product demand**.  

Using **SQL, Excel, and Power BI**, raw data was transformed into **interactive dashboards and actionable insights** to support data-driven business decisions.

---

## 🔍 Key Highlights  

### 🧹 Data Preparation  
✔ Cleaned and structured **1,000+ records** to ensure data accuracy and consistency  

### 🗄 SQL Analysis  
✔ Developed queries to extract KPIs such as:  
- Revenue trends  
- Return rates  
- Category performance  

### 📊 Dashboard Development  
Designed an **interactive Power BI dashboard** featuring:  
- 📈 Revenue trends & seasonality  
- 🌍 Regional and category-wise performance  
- 👥 Customer segmentation (loyalty tiers)  
- 🔄 Sales & return behavior analysis  

### 💡 Insight Generation  
- Identified **high-value customers (Platinum tier)**  
- Detected **peak sales months (May–June)**  
- Highlighted **27% return rate** as a key concern
